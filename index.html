1. git init - инициализирует работу с git
  2. git status - показывает статус, на какой ветке, какой файл модифицирован
  3. git add -a - помечает новые файли которые нужно добавить
  4. git commit -m 'message' - создает комит, точку модификации файла в который были изменения
  5. git pull origin main - создает копию репозитория на компьютере
  6. git push origin main - отправляет 'закомиченые' фалайлы в репозиторий
  7. git checkout -b main - переключаеться на требуемую ветку
  8. git branch 'name' - создает новую ветку
  9. git log - показывает все коммиты которые были сделаны в этой ветке
